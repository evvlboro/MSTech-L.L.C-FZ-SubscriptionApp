# 📋 План проекта: Subscription App с нуля

## 🛠 Технологии

**Основной стек:**
- **React Native** с **TypeScript** (вместо JS)
- **Expo** для упрощенной разработки
- **Expo Go** для тестирования на телефоне
- **React Navigation** для навигации
- **AsyncStorage** для локального хранения
- **Ionicons** для иконок

**Для деплоя:**
- **EAS Build** (Expo Application Services) - для билдов APK/IPA
- Или можно использовать **Expo Updates** для OTA обновлений
- Для веб-версии: `npx expo export:web`

## 🚀 GitHub: Пошаговая инструкция

### 1. Создание репозитория на GitHub

1. Заходишь на github.com
2. New repository
3. Название: `subscription-app`
4. Public/Private (на выбор)
5. **НЕ** добавляй README, .gitignore, license (создадим локально)
6. Create repository

### 2. Инициализация проекта и подключение к GitHub

```bash
# Создаем Expo проект с TypeScript
npx create-expo-app SubscriptionApp --template blank-typescript

# Переходим в папку
cd SubscriptionApp

# Инициализируем git (если не создан автоматически)
git init

# Создаем .gitignore (Expo уже создает его)
# Делаем первый коммит
git add .
git commit -m "Initial commit: Expo TypeScript project"

# Подключаем удаленный репозиторий (замени на свой URL)
git remote add origin https://github.com/username/subscription-app.git

# Устанавливаем ветку main
git branch -M main

# Пушим в GitHub
git push -u origin main
```

## 🔧 Сервисы

### subscriptionService.ts

**Промпт:**
```
"Создай сервис subscriptionService.ts для работы с подпиской на TypeScript:

Импорты:
- AsyncStorage из '@react-native-async-storage/async-storage'
- Тип Subscription из types

Константа:
- SUBSCRIPTION_KEY = '@subscription_status'

Методы:
1. getSubscription(): Promise<Subscription | null>
   - Получает данные из AsyncStorage
   - Возвращает null если нет данных
   - Обрабатывает ошибки (console.error)

2. saveSubscription(type: 'month' | 'year'): Promise<Subscription>
   - Создает объект { isActive: true, type, date: new Date().toISOString() }
   - Сохраняет в AsyncStorage
   - Возвращает созданный объект
   - Обрабатывает ошибки

3. cancelSubscription(): Promise<void>
   - Удаляет данные из AsyncStorage
   - Обрабатывает ошибки

Экспорт:
- export const subscriptionService = { getSubscription, saveSubscription, cancelSubscription }
"
```
